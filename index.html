OV7670-Explore

Some code for the OV7670 using an Arduino MEGA and a LCD/TFT QVGA display. It allows the exploration of the resolutions and the various modes of operation. For example - Brightness, Contrast, Black & White output and VGA "Windowing.

This code allows you to explore the functions built into the camera rather than load a canned set of register settings. The resolutions supported are: VGA, QVGA, QQVGA, CIF, QCIF, Q8060 and Q4030. The last two are the picture sized 80x60 and 40x30.
